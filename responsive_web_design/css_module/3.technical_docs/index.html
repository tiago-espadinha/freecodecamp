Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It is widely used in web development, data science, automation, artificial intelligence, and more.

Python emphasizes code readability and allows developers to write clear and logical code for both small and large-scale projects.

Some key features of Python:

  • Easy to learn and use
  • Large standard library
  • Cross-platform compatibility
  • Strong community support
What You Should Already Know

Before starting with Python, you should have:

  • Basic understanding of how computers work
  • Familiarity with programming concepts (optional but helpful)
  • Basic knowledge of text editors or IDEs
Python vs Other Languages

Python differs from many languages like Java or C++:

  • It uses dynamic typing instead of static typing
  • It has a simpler, more readable syntax
  • It does not require compilation before execution
  • It uses indentation instead of braces

Python:


        if x > 5: 

            print("Hello")

C++:


        if (x > 5) { 

            cout << "Hello";

        }
Hello World

Your first Python program:


        print("Hello, World!")

This prints text to the console.

Variables

Variables store data values.

Example:


        x = 10 

        name = "Alice"

Rules:

  • No declaration keyword needed
  • Type is assigned automatically
  • Case-sensitive
Data Types

Common Python data types:

  • Integer → 10
  • Float → 3.14
  • String → "Hello"
  • Boolean → True / False
  • List → [1, 2, 3]
  • Dictionary → {"key": "value"}

Example:


        age = 25 

        price = 19.99 

        is_valid = True
Control Flow

Python uses conditional statements to control execution.

Example:

if x > 10: print("Greater than 10") elif x == 10: print("Equal to 10") else: print("Less than 10")
Loops

While Loop


        count = 0 

        while count < 5: 

            print(count) 

            count += 1

For Loop


        for i in range(5):

            print(i)
Functions

Functions allow code reuse.

Example:


        def greet(name): 

            return "Hello " + name

Calling a function:


        print(greet("Alice"))
Modules

Modules are files containing Python code.

Example:


        import math 

        print(math.sqrt(16))

You can also import specific functions:


        from math import sqrt
File Handling

Python can read and write files.

Write to a file:


        with open("file.txt", "w") as f: 

            f.write("Hello World")

Read a file:

 
        with open("file.txt", "r") as f: 

            print(f.read())
Error Handling

Handle errors using try-except:


        try: 

            x = int("abc") 

        except ValueError: 

            print("Conversion failed")
